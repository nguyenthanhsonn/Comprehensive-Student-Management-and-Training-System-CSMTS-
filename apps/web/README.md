# Web App

Thu muc nay danh cho Next.js frontend.

- `src/app/(auth)`: cac man hinh dang nhap va xac thuc.
- `src/app/(dashboard)`: cac module nghiep vu sau khi dang nhap.
- `src/components`: thanh phan giao dien tai su dung.
- `src/lib`: helpers cho API, auth va utility.
- `src/hooks`, `src/types`, `src/styles`: hook, kieu du lieu va style dung chung.
