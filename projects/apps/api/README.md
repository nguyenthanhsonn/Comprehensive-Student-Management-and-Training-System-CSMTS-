# API App

Thu muc nay danh cho NestJS REST API.

- `src/common`: guard, pipe, interceptor, filter va decorator dung chung.
- `src/config`: cau hinh environment va validation.
- `src/database`: ket noi database, Prisma adapter va seed support.
- `src/modules`: cac module nghiep vu map truc tiep tu docs.
